# MasSubCollector

## Description:
By using the following tools
[subfinder, assetfinder, findomain]
in an automated manner to input them and gather subdomains.
After that, the subdomains are saved inside files with the same names as the used tools.


To check the requirements, run the following command
```
chmod +x Check.sh && ./Check.sh
```

To run the script with a list of domains, use the following command: 
```
chmod +x mscollector.sh
./mscollector.sh -f domains-list.txt 
```
